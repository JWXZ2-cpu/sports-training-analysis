# 脉 MAI — AI 运动训练分析平台

## 项目介绍

**脉（MAI）** 是一套面向专业运动队的 AI 训练分析平台，覆盖运动员、主教练、助教、队医、管理人员五大角色，通过"语音/文字反馈 → AI 分析 → 角色化分发"的核心数据流，将训练反馈、身体状态、心理评估整合为结构化的决策支持。

系统将丹尼尔斯跑步训练法、邦帕周期化训练理论和运动心理学三大经典理论内嵌至 AI 引擎，实现从微观配速到宏观周期、从身体状态到心理状态的全维度分析。

### 核心特色

- **双视角智能输出** — 运动员看到训练日记，教练看到专业分析，避免"AI 比教练更懂"的信任危机
- **语音驱动低门槛交互** — 运动员训练后按住说话即可完成反馈，AI 自动解析为结构化数据
- **初始能力评估** — 首次登录通过 PB 成绩或 Conconi 测试计算 VDOT，建立个人化训练区间
- **治疗-训练冲突检测** — 队医录入治疗后自动与次日训练计划交叉比对，提前发现风险
- **多团队隔离** — 邀请码机制加入团队，数据按团队严格隔离
- **三语国际化** — 中文 / English / Italiano 无缝切换

### 技术栈

| 层级 | 技术 |
|---|---|
| 前端 | React 19 + Vite + React Router 7 |
| 后端 | Express 5 + 自研 JsonDB |
| AI | MIMO mimo-v2.5（分析）+ mimo-v2.5-asr（语音识别）|
| 认证 | JWT + RBAC 角色权限 |
| 部署 | Node.js + PM2 |

### 角色功能

| 角色 | 核心能力 |
|---|---|
| 🏃 运动员 | 语音/文字反馈、训练日记、FIT 数据导入、个人趋势、体测管理 |
| 🧑‍🏫 主教练 | 团队总览、AI 辅助制定计划、训练课录入、绑定管理 |
| 👨‍💼 助教 | 全队数据查看、训练备注、报告翻译 |
| 🩺 队医 | 伤病/治疗管理、医疗文档上传、治疗-训练冲突检查 |
| 📊 管理人员 | 考勤统计、训练完成率、团队状态看板 |

### 语音功能说明

语音录音需要浏览器麦克风权限，**要求 HTTPS 环境**：
- 电脑端 Chrome/Edge：localhost 自动支持，生产环境需 HTTPS
- 手机端：必须通过 HTTPS 访问（微信内置浏览器不支持，需用浏览器打开）
- 无 HTTPS 时自动降级为文字输入模式
